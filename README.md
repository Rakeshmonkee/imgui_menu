# imgui_menu

A basic DXD9 IMGUI menu I made for Rust that I never completed.

VCPKGS:
- Nlohmann JSON: `vcpkg install nlohmann-json:x64-windows`
- GLFW OpenGL: `vcpkg install glfw3:x64-windows`
